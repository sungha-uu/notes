# 당뇨식단 시나리오룰.md
> 채널: 당뇨식단 유튜브 채널 | 유형 코드: `DIABETES_DIET`

---

## 1. 콘텐츠 목적
당뇨 환자 및 혈당 관리가 필요한 사람들을 위한 식단 정보와 레시피를 제공한다.
의학적 정보를 쉽게 풀어내어 "맛있게, 안전하게" 먹을 수 있는 방법을 안내한다.

> ⚠️ **필수 면책 고지**: 모든 영상에 "이 콘텐츠는 정보 제공 목적이며 의료 조언을 대체하지 않습니다. 반드시 담당 의사·영양사와 상담하세요" 문구를 영상 시작 또는 말미에 포함.

---

## 2. 콘텐츠 분류

| 카테고리 | 세부 내용 | 비고 |
|----------|----------|------|
| 혈당 지수 정보 | 식품별 GI 지수, 혈당 올리지 않는 식품 | 의료 출처 필수 |
| 당뇨 레시피 | 저당 식사, 간식, 도시락 | 칼로리·당 함량 필수 표기 |
| 식품 비교 | 먹어도 되는 음식 vs 피해야 할 음식 | 대비 구성 |
| 외식 가이드 | 식당에서 혈당 안 올리는 주문법 | 실용적 팁 |
| 혈당 관리 팁 | 식사 순서, 식후 운동, 포션 조절 | 생활 습관 |

## 3. 수집 소스 (신뢰도 최우선)

### 1순위 — 의료·과학 기관 (필수)
- 대한당뇨병학회 (diabetes.or.kr)
- 국민건강보험공단 건강 정보
- 질병관리청 (kdca.go.kr)
- 서울대병원·삼성서울병원·서울아산병원 당뇨 정보 페이지
- 한국영양학회 (nutrition.or.kr)

### 2순위 — 전문가 검토 콘텐츠
- 내분비내과 전문의 유튜브 채널
- 공인 영양사(RD) 작성 블로그 또는 콘텐츠
- 당뇨 전문 병원 공식 블로그

### 3순위 — 보조 참고 (검증 후 사용)
- 당뇨 환자 커뮤니티 (당뇨카페 등) — 경험담만, 의학 정보로 사용 금지
- 해외 자료: ADA (American Diabetes Association), NHS (영국)

### 절대 사용 금지 소스
- 개인 블로그의 "당뇨 완치" 주장 콘텐츠
- 건강기능식품 판매 사이트의 정보
- 근거 없는 민간요법 관련 자료
- 출처 불명 SNS 게시물

## 4. 수집 필터 규칙

### 포함 기준
- 반드시 공공·의료기관 출처 명시
- 혈당지수(GI) 또는 혈당부하지수(GL) 수치 제공 가능한 정보
- 최근 5년 이내 발표된 자료
- 한국인 식단 기준에 맞는 정보 (서양식 단독 자료 사용 금지)

### 제외 기준
- "당뇨 완치", "혈당 즉시 정상화" 등 과장 주장
- 특정 건강기능식품·보조제 홍보
- 의학적 근거 없는 단식·극단적 식이 요법
- 수치 없이 "좋다/나쁘다"만 서술한 정보

---

## 5. 영상 1편 구성 규칙

| 항목 | 규정 |
|------|------|
| 영상 길이 | 2분 ~ 3분 |
| 구성 흐름 | 문제 제기(15초) → 핵심 정보 제시(90~120초) → 실천 팁(20초) → 면책 고지 + CTA(15초) |
| 제목 형식 | `당뇨 있어도 먹을 수 있는 {음식} \| 혈당 안 올리는 방법` 또는 `[당뇨식단] {주제} 완벽 가이드` |
| 썸네일 | 음식 사진 + "혈당 OK" 또는 "GI지수 N" 텍스트 뱃지 |

---

## 6. 스크립트 작성 규칙

- 어투: 따뜻하고 이해하기 쉬운 설명형 (환자 눈높이)
- 수치 반드시 포함 (GI 지수, 칼로리, 당 함량 등)
- "~할 수 있습니다", "~로 알려져 있습니다" 등 신중한 표현 사용
- 금지 표현: "드시면 혈당이 내려갑니다", "완치됩니다", "무조건 좋습니다"
- 필수 포함 문구: "개인 차이가 있으므로 담당 의료진과 상담하세요"
- 출처를 자막 또는 영상 설명에 반드시 명시

## 7. 영상 프롬프트 생성 규칙

```
유형: 당뇨식단
주제: {주제명}
배경 스타일: 깔끔한 의료·건강 느낌, 화이트+그린 계열, 인포그래픽 스타일
영상 요소: 인포그래픽 (GI 지수 시각화), 음식 사진, 식품 비교표, 아이콘
텍스트 스타일: 명확하고 가독성 높은 고딕, 수치 강조 (초록: 좋음, 빨강: 주의)
나레이션 톤: 차분하고 신뢰감 있는 목소리, 너무 빠르지 않게
배경음악: 차분하고 집중되는 ambient (Mubert: health wellness 태그)
전환 효과: 인포그래픽 등장 애니메이션, 깔끔한 슬라이드
말미 구성: 면책 고지 자막 + 출처 표기
```

## 8. 품질 체크리스트

- [ ] 의료·공공기관 출처가 최소 1개 이상 포함되었는가
- [ ] 과장 표현이 없는가
- [ ] 수치 정보(GI, 칼로리 등)가 포함되었는가
- [ ] 면책 고지 문구가 영상에 포함되었는가
- [ ] 한국인 식단 기준에 맞는 정보인가
- [ ] 특정 제품 홍보 내용이 없는가
