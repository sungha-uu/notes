# 육아팁 시나리오룰.md
> 채널: 육아팁 유튜브 채널 | 유형 코드: `PARENTING`

---

## 1. 콘텐츠 목적
개월수별로 아기 발달 단계에 맞는 실용적인 육아 팁을 제공한다.
부모가 "지금 우리 아이한테 딱 맞는 정보"를 찾을 수 있도록 구체적이고 신뢰도 높은 내용만 수록한다.

---

## 2. 수집 대상 개월수 구간

| 구간 | 범위 | 키워드 예시 |
|------|------|------------|
| 신생아기 | 0~3개월 | 수유 간격, 수면 패턴, 황달, 배꼽 관리 |
| 초기 영아기 | 4~6개월 | 이유식 시작, 뒤집기, 낯가림 시작 |
| 중기 영아기 | 7~9개월 | 기기, 이앓이, 손가락 음식, 분리불안 |
| 후기 영아기 | 10~12개월 | 첫 걸음마, 언어 발달, 젖병 떼기 |
| 초기 유아기 | 13~18개월 | 자아 발달, 떼쓰기 시작, 소근육 놀이 |
| 중기 유아기 | 19~24개월 | 화장실 훈련, 두 단어 문장, 형제 갈등 |
| 후기 유아기 | 25~36개월 | 어린이집 적응, 감정 조절, 독립심 키우기 |

## 3. 수집 소스 (우선순위 순)

### 1순위 — 공공기관 및 의료기관
- 대한소아과학회 (pediatrics.or.kr)
- 질병관리청 예방접종 정보
- 서울대학교병원 / 삼성서울병원 육아 콘텐츠
- 보건복지부 임신육아종합포털 아이사랑 (childcare.go.kr)

### 2순위 — 검증된 전문가 콘텐츠
- 소아과 전문의 유튜브 채널 (하정훈 원장 등)
- 국내 육아 전문 출판사 블로그 (키즈맘, 베이비뉴스)
- 국제 기준: WHO, AAP (American Academy of Pediatrics) 가이드라인

### 3순위 — 커뮤니티 (보조 참고만)
- 맘카페 (네이버 맘스홀릭베이비 등) — 실제 경험담 수집용
- 육아 앱 리뷰 (아이챙겨, 베이비타임)
- ※ 커뮤니티 정보는 단독 사용 금지, 반드시 1·2순위 출처로 검증 후 사용

---

## 4. 수집 필터 규칙

### 포함 기준
- 해당 개월수 범위 명시된 정보
- 국내 육아 환경 기준 (한국 예방접종 스케줄, 한국 이유식 문화 등)
- 최근 3년 이내 발행 자료 우선
- 소아과 전문의 검토 또는 공공기관 출처 명시된 자료

### 제외 기준
- 출처 불명 SNS 게시물
- 의학적 근거 없는 민간요법 단독 제시
- 해외 기준만 적용된 정보 (예: 미국 기준 이유식 시작 월령)
- 특정 제품·브랜드 홍보 목적 콘텐츠
- 5년 이상 된 구식 정보 (예: 배앓이에 보리차 권장 등)

## 5. 영상 1편 구성 규칙

| 항목 | 규정 |
|------|------|
| 영상 길이 | 1분 30초 ~ 2분 30초 |
| 팁 개수 | 1편당 3~5개 팁 |
| 구성 흐름 | 훅(10초) → 개월수 소개(10초) → 팁 나열(60~90초) → 마무리 CTA(10초) |
| 제목 형식 | `[N개월 아기] 꼭 알아야 할 육아 팁 N가지` |
| 썸네일 키워드 | 개월수 숫자 크게, 핵심 팁 키워드 1개 |

---

## 6. 스크립트 작성 규칙

- 호칭: "아기", "우리 아이", "아가" 혼용 (너무 반복 금지)
- 어투: 친근하고 따뜻한 구어체 (예: "이 시기엔 이렇게 해주세요~")
- 전문 용어 사용 시 반드시 쉬운 설명 병행
- 문장 길이: 나레이션 기준 1문장 = 최대 15자 이내 권장
- 금지 표현: "무조건", "절대로", "100% 효과" 등 과장 표현
- 필수 포함: 영상 말미에 "개인차가 있으니 소아과 전문의와 상담하세요" 문구

## 7. 영상 프롬프트 생성 규칙 (영상 에이전트 전달용)

```
유형: 육아팁
개월수: {N}개월
핵심 팁: {팁1}, {팁2}, {팁3}
배경 스타일: 파스텔 톤, 따뜻한 아이보리·민트 계열
등장 요소: 귀여운 아기 일러스트 or 실제 아기 영상 클립
텍스트 스타일: 둥근 폰트, 강조 키워드는 노란 하이라이트
나레이션 톤: 부드럽고 따뜻한 여성 목소리 (ElevenLabs: Rachel 또는 한국어 전용 모델)
배경음악: 잔잔한 피아노 or 오르골 (Mubert: parenting 태그)
전환 효과: 부드러운 페이드, 빠른 컷 금지
```

## 8. 품질 체크리스트 (메인 에이전트 검수용)

- [ ] 개월수가 제목·영상에 명확히 표시되는가
- [ ] 출처가 1·2순위 소스로만 구성되었는가
- [ ] 팁이 3개 이상 포함되었는가
- [ ] 과장 표현이 없는가
- [ ] 전문의 상담 권고 문구가 포함되었는가
- [ ] 영상 길이가 1분 30초 ~ 2분 30초 범위인가
