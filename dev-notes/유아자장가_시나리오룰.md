# 유아 자장가 시나리오룰.md
> 채널: 유아 자장가 유튜브 채널 | 유형 코드: `LULLABY`

---

## 1. 콘텐츠 목적
0~5세 영유아를 재우기 위한 자장가 및 수면 유도 음악 영상을 제공한다.
부모가 틀어놓으면 아이가 편안하게 잠들 수 있는 영상 환경을 만든다.

---

## 2. 콘텐츠 분류

| 카테고리 | 세부 내용 | 영상 길이 |
|----------|----------|----------|
| 전통 자장가 | 우리나라 전래 자장가 (자장자장, 섬집아기 등) | 10~30분 |
| 클래식 자장가 | 브람스, 모차르트 자장가 등 | 10~30분 |
| 창작 자장가 | AI 또는 저작권 프리 신규 자장가 | 10~20분 |
| 자연 소리 + 자장가 | 빗소리, 파도소리, 모닥불 + 자장가 믹스 | 30~60분 |
| 수면 백색소음 | 헤어드라이어, 자궁 소리, 선풍기 소리 (신생아 특화) | 30~60분 |
| 명상형 수면 | 부드러운 피아노 + 자연 소리 | 30~60분 |

---

## 3. 수집 소스 및 음원 기준

### 음원 소스 (저작권 필수 확인)
- **전통 자장가**: 저작권 소멸된 전래 자장가만 사용 (창작 연도 50년 이상)
- **클래식**: 퍼블릭 도메인 악보 기반 — IMSLP (imslp.org) 확인
- **창작/배경음악**: 
  - Suno AI (생성 음악, 상업 라이선스 확인)
  - Mubert (royalty-free, 상업 사용 가능 플랜)
  - Pixabay Music (CC0 라이선스)
  - YouTube Audio Library (저작권 프리 필터)

### 절대 사용 금지
- 멜론·지니·벅스 등 상업 음원 플랫폼 수록 곡
- 저작권자 확인 안 된 유튜브 자장가 영상 재사용
- 아티스트 이름 있는 현대 자장가 앨범 (반드시 라이선스 확인)

---

## 4. 수집 필터 규칙

### 포함 기준
- 저작권이 명확히 해결된 음원만
- 60~80 BPM 이하의 느린 템포 (수면 유도 효과)
- 볼륨이 갑자기 튀는 구간 없음 (급격한 변화 금지)
- 영유아에게 안전한 음량 범위 (마스터링 -14 LUFS 이하)

### 제외 기준
- 템포 빠른 동요 (곰 세 마리, 머리 어깨 무릎 발 등)
- 가사에 무서운 내용 포함된 자장가 ("호랑이가 잡아간다" 류)
- 갑작스러운 효과음 포함 음원
- 저작권 불분명한 음원

---

## 5. 영상 1편 구성 규칙

| 항목 | 규정 |
|------|------|
| 영상 길이 | 10분 (짧은 버전) / 30분 / 60분 (긴 버전) |
| 시각 요소 | 움직임 최소화 — 별, 달, 구름, 잠자는 동물 등 부드러운 애니메이션 |
| 화면 밝기 | 어둡고 따뜻한 톤 (수면 유도용, 블루라이트 최소화) |
| 자막 | 자장가 가사 (있는 경우) 부드럽게 표시 or 자막 없음 |
| 제목 형식 | `[자장가 N분] {음악명} \| 아기 수면 유도 \| 0~5세` |
| 썸네일 | 달빛 아래 잠자는 아기 일러스트, 어두운 파란색 배경 |

---

## 6. 스크립트 작성 규칙

자장가 영상은 나레이션이 없거나 최소한으로 유지:
- 영상 시작 시 짧은 인트로 멘트만 허용 (예: "이제 잠잘 시간이에요. 편안하게 눈을 감아요.")
- 이후 음악만 진행
- 챕터 마커로 곡 구분 (긴 버전의 경우)

---

## 7. 영상 프롬프트 생성 규칙

```
유형: 유아 자장가
카테고리: {전통/클래식/창작/자연소리/백색소음/명상형}
음원: {음원명 및 출처}
배경 스타일: 어둡고 따뜻한 밤하늘, 별, 달, 구름, 잠자는 동물 (고양이, 토끼, 곰 등)
색상: 네이비, 딥퍼플, 소프트골드 계열 — 블루라이트 최소화
애니메이션: 아주 느리고 부드러운 움직임 (반짝이는 별, 천천히 흐르는 구름)
텍스트: 최소화 — 제목만 초반에 부드럽게 표시 후 사라짐
나레이션: 없음 or 아주 짧은 인트로만 (속삭이는 톤)
배경음악: 선택된 자장가 음원 그대로 사용
화면 밝기: 50% 이하 어두운 톤 유지
```

---

## 8. 품질 체크리스트

- [ ] 사용된 모든 음원의 저작권이 해결되었는가
- [ ] 갑작스러운 소리 변화가 없는가
- [ ] 화면이 충분히 어둡고 따뜻한 톤인가
- [ ] 영상 길이가 계획한 버전 (10/30/60분) 기준에 맞는가
- [ ] 아이들에게 불안감을 줄 수 있는 이미지가 없는가
- [ ] 음량 마스터링이 안전 범위 (-14 LUFS 이하)인가
