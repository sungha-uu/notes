---

## 1. 콘텐츠 목적
0~7세 유아를 위한 한국 전래동화를 쉽고 재미있게 영상으로 제작한다.
아이들이 우리 문화와 가치관을 자연스럽게 익힐 수 있도록 교육적 메시지를 담는다.

---

## 2. 전래동화 분류 체계

### 인물 유형별
| 유형 | 대표 이야기 예시 |
|------|----------------|
| 동물 주인공 | 토끼와 자라, 호랑이와 곶감, 여우와 두루미 |
| 인간 주인공 | 콩쥐팥쥐, 흥부와 놀부, 선녀와 나무꾼 |
| 요괴/신비 | 도깨비 이야기, 용왕이야기, 삼신할머니 |
| 지혜/꾀 | 방귀쟁이 며느리, 재치 있는 아이, 바보 온달 |

### 교육 테마별 (1편 = 1테마)
| 테마 | 이야기 예시 | 교훈 |
|------|-----------|------|
| 나눔과 배려 | 흥부와 놀부 | 욕심 금지, 나눔의 기쁨 |
| 정직 | 토끼의 간 | 거짓말의 결과 |
| 효도 | 심청전 | 부모님 사랑 |
| 용기 | 해와 달이 된 오누이 | 위험 앞의 용기 |
| 지혜 | 혹부리 영감 | 지혜로운 판단 |
| 친구 우정 | 의좋은 형제 | 우정, 협력 |

## 3. 전래동화 선정 기준

### 포함 기준
- **저작권 소멸 확인**: 전통 구전 전래동화 (창작 연도 불명 또는 저작권 소멸)
- 유아 연령 (0~7세)에 적합한 이야기 복잡도
- 교육적 메시지가 명확한 이야기
- 폭력·공포 장면을 순화하여 표현 가능한 이야기
- 한국 문화 특색이 담긴 이야기 우선

### 제외 기준
- 현대 창작 동화 (저작권 문제)
- 순화 불가능한 수위의 폭력·공포 장면 포함 이야기
- 특정 종교·이념 편향된 이야기
- 성별 고정관념이 지나치게 강한 이야기 (시대에 맞게 각색 필요 시 별도 표기)

## 4. 수집 소스

### 1순위 — 공공 기관 아카이브
- 국립민속박물관 한국민속대백과사전 (folkency.nfm.go.kr)
- 한국학중앙연구원 한국구비문학대계
- 문화재청 무형문화재 관련 자료
- 국립어린이청소년도서관 추천 전래동화 목록

### 2순위 — 검증된 출판 자료
- 출판사 전래동화 시리즈 (웅진, 비룡소, 길벗어린이 등) — 내용 참고만, 그대로 사용 금지
- 교육부 인정 교재 수록 전래동화

### 창작 각색 지침
- 원전 이야기를 바탕으로 영상용 스크립트로 재창작
- 어려운 고어·사투리는 현대어로 순화
- 폭력적 장면은 암시적으로 표현 (예: "나쁜 호랑이가 혼이 났어요" 수준)
- 각색 시 원전 출처 명시

## 5. 영상 1편 구성 규칙

| 항목 | 규정 |
|------|------|
| 영상 길이 | 3분 ~ 5분 (유아 집중력 기준) |
| 구성 흐름 | 오프닝 징글(10초) → 이야기 소개(15초) → 본 이야기(2~3분) → 교훈 정리(30초) → 엔딩(15초) |
| 제목 형식 | `[전래동화] {이야기 제목} \| 유아 동화 \| 교훈: {교훈 키워드}` |
| 썸네일 | 이야기 주인공 일러스트 + 제목, 밝고 귀여운 스타일 |
| 시리즈 구성 | 연관 이야기 시리즈로 묶기 (예: "동물 주인공 전래동화 시리즈") |

---

## 6. 스크립트 작성 규칙

- 어투: 할머니·할아버지가 이야기해주는 따뜻한 구어체
- 문장 길이: 짧고 리드미컬하게 (유아 청취 기준)
- 의성어·의태어 적극 활용 (예: "덩실덩실", "깡충깡충", "쿵쾅쿵쾅")
- 감정 강조 표현 사용 (나레이터가 캐릭터처럼 표현)
- 각 장면 전환 시 "그런데 말이에요~", "어느 날~" 등 자연스러운 연결어 사용
- 교훈은 억지스럽지 않게, 자연스럽게 마무리
- 어린이 청자 직접 호명 가능 (예: "얘들아, 이 이야기 들어봤니?")
- 금지 표현: 공포를 직접적으로 조장하는 표현, 폭력 묘사 직접 언급

## 7. 영상 프롬프트 생성 규칙

```
유형: 유아 전래동화
이야기 제목: {제목}
주인공: {캐릭터명}
교훈 테마: {테마}
배경 스타일: 한국 전통 분위기 + 귀여운 동화 일러스트 (수채화 or 2D 애니메이션 느낌)
색상 팔레트: 따뜻하고 밝은 색상 (옅은 황금, 하늘색, 연두 계열) — 무섭지 않게
캐릭터 스타일: 둥글둥글 귀여운 형태, 표정 풍부하게
배경 요소: 한옥, 산, 논밭, 한복 등 한국 전통 요소 자연스럽게 포함
텍스트 스타일: 동글동글 귀여운 폰트, 주요 단어만 강조
나레이션 톤: 따뜻하고 표현력 풍부한 목소리, 캐릭터별 목소리 변화
배경음악: 전통 악기 (가야금, 해금) 기반 동화풍 음악 (Mubert: kids storytelling 태그)
전환 효과: 부드러운 페이드 or 책 넘기는 효과
오프닝: 고정 채널 징글 (3~5초) → 이야기 시작
```

## 8. 콘텐츠 달력 (월별 기획 가이드)

| 월 | 추천 테마 | 이유 |
|----|----------|------|
| 1월 | 설날 관련 전래동화 | 명절 시즌 |
| 3월 | 봄·새 시작 이야기 | 새 학기 |
| 5월 | 가족·효도 이야기 | 어린이날·어버이날 |
| 8월 | 여름 배경 이야기 (도깨비 등) | 방학 시즌 |
| 10월 | 가을·추수 관련 이야기 | 추석 시즌 |
| 12월 | 겨울 배경 이야기 | 연말 시즌 |

## 9. 품질 체크리스트

- [ ] 원전 전래동화 기반이며 출처가 확인되었는가
- [ ] 유아(0~7세)에게 적합한 내용인가
- [ ] 폭력·공포 장면이 적절히 순화되었는가
- [ ] 교훈 메시지가 자연스럽게 전달되는가
- [ ] 한국 전통 문화 요소가 시각적으로 포함되었는가
- [ ] 영상 길이가 3~5분 범위인가
- [ ] 캐릭터 스타일이 일관성 있게 유지되는가
