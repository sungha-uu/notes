# CH03 — 건강관리 팁
> 채널 코드: `CH03_HEALTH`
> 채널 후보명: "1분 건강" / "오늘의 건강 1분" / "알면 다른 건강 습관"
> 대상 시청자: 전 연령 (타겟 세그먼트별 구분 운영)
> 영상 길이: 55초 ~ 1분 20초
> 업로드 주기: 주 3회
> 이미지 장수: 영상 1편당 9~11장

## 2. 세그먼트별 주요 주제 목록

### SEG_SENIOR — 노인 건강
```
관절 & 근골격계  : 관절염 완화 운동, 무릎 통증 줄이는 생활 습관, 낙상 예방
눈 건강         : 백내장 초기 증상, 황반변성 예방식품, 눈 건강 생활 수칙
뇌 & 인지       : 치매 예방 습관, 기억력 유지 식품, 두뇌 자극 활동
순환기          : 노인 고혈압 관리, 뇌졸중 전조 증상, 심장 건강 운동
근력 & 낙상     : 집에서 하는 하체 근력 운동, 균형 감각 향상 운동
```

### SEG_ADULT — 40~50대 중장년 건강
```
당뇨 관리       : 혈당 안 올리는 식단, 당뇨 환자 간식 추천, 혈당 조절 생활 습관
콜레스테롤      : LDL 낮추는 식품, 콜레스테롤 관리 운동, 피해야 할 음식
체중 & 비만     : 성인 비만 개선 식단, 내장지방 줄이는 운동, 야식 대처법
갱년기          : 갱년기 증상 완화 식품, 호르몬 건강 생활 수칙
간 & 소화기     : 지방간 개선 식단, 역류성 식도염 생활 습관, 장 건강 유지
직장인 건강     : 목·어깨 통증 스트레칭, 눈 피로 해소법, 앉아서 하는 운동
```

### SEG_KID — 아동 건강
```
감염병 & 바이러스: 독감 vs 감기 차이, 수족구병 대처법, RSV 바이러스 정보
예방접종        : 연령별 필수 예방접종 일정, 독감 예방접종 시기, 백신 부작용 대처
면역력          : 아이 면역력 높이는 식품, 손씻기 올바른 방법, 충분한 수면의 중요성
성장 & 영양     : 성장기 영양 섭취, 편식 개선 방법, 칼슘 & 철분 풍부한 아이 식품
```

### SEG_ALL — 전 연령 공통
```
수면            : 숙면을 위한 습관, 수면 부족의 신체 영향, 최적 수면 시간
스트레스        : 간단한 스트레스 해소법, 명상 기초, 호흡법
건강식 & 식재료  : 슈퍼푸드 효능, 계절별 건강 식품, 항산화 식품 TOP 5
기초 건강 습관  : 하루 물 섭취량, 자외선 차단의 중요성, 올바른 자세
```

## 4. 카테고리별 스크립트 구성

### DIET — 건강 식단
```
구성 흐름:
  00~10초 : 훅 — 대상 질환/목표 제시
             예) "당뇨 환자, 이 음식 먹으면 혈당 안 올라요"
  10~15초 : 왜 중요한지 한 줄 이유
  15~55초 : 식품/식단 3~5가지 구체적 설명 (식품명 + 이유 + 섭취법)
  55~70초 : 주의사항 + 의료진 상담 권고

필수 포함:
  - 수치 기반 정보 (예: "하루 30g 이내", "GI 지수 55 이하")
  - 구체적 식품명 (막연히 "채소" 금지 → "시금치, 브로콜리, 양배추")
  - 피해야 할 음식 1~2가지 함께 제시
```

### EXER — 집에서 하는 운동법
```
구성 흐름:
  00~08초 : 이 운동이 필요한 사람 / 효과 제시
  08~55초 : 동작 2~4가지 (동작명 + 횟수/시간 + 주의사항)
  55~70초 : 운동 빈도 권장 + 하면 안 되는 사람 안내

필수 포함:
  - 횟수/시간 반드시 수치로 명시 ("하루 10분", "10회 3세트")
  - SENIOR 대상: 낙상 주의, 무리하지 말 것 강조
  - 운동 전 준비 동작 언급
  - "통증이 느껴지면 즉시 중단하세요" 문구 필수
```

### DISEASE — 질환 정보 & 예방법
```
구성 흐름:
  00~10초 : 질환 또는 증상 소개 (얼마나 흔한지 수치 제시)
  10~45초 : 증상 설명 + 예방 습관 3가지
  45~60초 : 병원 가야 할 신호 + 정기 검진 권고

필수 포함:
  - 유병률 or 통계 수치 (신뢰도 확보)
  - "이런 증상이 있으면 즉시 병원" 명확히 제시
  - 정기 검진 주기 안내
  - "진단과 치료는 반드시 전문의와 상담하세요" 면책 고지
```

### VIRUS / VACCINE — 감염병 & 예방접종
```
구성 흐름:
  00~10초 : 현재 유행 상황 or 예방접종 시기 안내
  10~50초 : 증상 / 접종 일정 / 예방법 구체적 설명
  50~65초 : 고위험군 주의사항 + 의료기관 방문 권고

필수 포함:
  - 질병관리청 최신 발표 기준 (수집 날짜 명시)
  - 대상 연령 / 고위험군 명확히 구분
  - 예방접종 무료 대상 여부 안내
  - "정확한 접종 일정은 소아과/가정의학과에 확인하세요"
```

## 5. 공통 스크립트 규칙

```
면책 고지 (필수):
  모든 영상 말미에 반드시 포함
  → "이 콘텐츠는 건강 정보 제공 목적이며,
     진단·치료는 반드시 전문의와 상담하세요."

톤 기준:
  SEG_SENIOR  : 천천히 또렷하게, 어려운 용어 최소화, 따뜻하고 안심시키는 톤
  SEG_ADULT   : 실용적이고 직접적으로, "지금 당장 바꿀 수 있는" 느낌
  SEG_KID     : 부모에게 전달하는 톤, 걱정을 덜어주는 안심형 어조
  SEG_ALL     : 친근하되 신뢰감 있게, 생활 밀착형

문장 길이   : TTS 1문장 = 20자 이내 권장
전문 용어   : 사용 시 즉시 쉬운 표현 병기
              예) "LDL 콜레스테롤 → 나쁜 콜레스테롤이라고 불리는"
수치 필수   : 막연한 표현 금지
              "적게" → "하루 2,000mg 이하로"
              "충분히" → "하루 30분 이상"
금지 표현   : "완치됩니다", "드시면 낫습니다", "무조건 효과 있습니다"
              "이것만 먹으면", "기적의 식품"
```

### 장면별 구성 (1편 = 10장 기준)
```
01장 : 제목 카드 — 세그먼트 + 주제 텍스트 삽입용 깔끔한 배경
02장 : 훅 장면 — 대상 인물 + 상황 묘사 (해당 질환/목표 연상)
03장 : 문제 제기 카드 — 통계 or 현황 텍스트 삽입용 공간
04~08장 : 본문 설명 장면 (식품/동작/정보 시각화, 텍스트 삽입 공간 확보)
09장 : 핵심 요약 카드 — 오늘의 핵심 메시지 텍스트 삽입용
10장 : 면책 고지 카드 — "전문의 상담 필수" 문구 삽입용 차분한 배경
```

## 9. 제목 & 썸네일 공식

```
제목 형식:
  패턴 A — "[당뇨/관절염/etc] {핵심 정보} {숫자}가지 | {부제}"
  패턴 B — "{대상}을 위한 {주제} | 1분 건강 정보"
  패턴 C — "몰랐던 {건강 사실} | 지금 당장 바꿔야 할 습관"

예시:
  - [당뇨 식단] 혈당 안 올리는 간식 5가지 | 집에서 바로 실천
  - 50대 콜레스테롤 관리법 | LDL 낮추는 식품 TOP 4
  - 관절염 있어도 할 수 있는 실내 운동 3가지
  - 독감 유행 시즌, 아이 면역력 지키는 방법 4가지
  - 잠 잘 못 자면 생기는 일 | 숙면 습관 3가지

썸네일 구성:
  - 배경: 화이트 + 세그먼트별 포인트 컬러
  - 중앙: 핵심 키워드 굵은 폰트 크게
  - 인물/식품/아이콘 일러스트 배치
  - 뱃지: 대상 세그먼트 표기 (예: "40~50대", "노인 건강", "아이 건강")
  - 포인트: 초록색 or 파란색 강조선
```
