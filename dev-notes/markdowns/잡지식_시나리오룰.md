# 잡지식_시나리오룰.md
> 채널: 알아두면 쓸데없는 잡지식 유튜브 채널 | 유형 코드: `TRIVIA`

---

## 1. 콘텐츠 목적
"알고 나면 써먹을 데 없지만 알고 싶어지는" 흥미로운 잡학 지식을 제공한다.
시청자가 "오 진짜? 신기하다!" 반응이 나오는 콘텐츠 — 지식 자체보다 신기함과 재미가 핵심.

---

## 2. 지식 분류 체계

| 카테고리 | 예시 주제 |
|----------|----------|
| 과학 잡지식 | 왜 종이는 32번 이상 못 접나, 파란 눈이 생긴 이유 |
| 역사 잡지식 | 고대 로마의 치약은 오줌이었다, 나폴레옹의 실제 키 |
| 언어/어원 | "망했다"의 어원, 영어 단어 중 가장 긴 단어 |
| 동물 잡지식 | 문어는 심장이 3개다, 달팽이는 이빨이 2만 개 |
| 일상의 비밀 | 왜 하품은 전염될까, 영화관 팝콘이 비싼 이유 |
| 숫자 잡지식 | 원래 10월(October)이 8번째 달이었던 이유 |
| 한국 잡지식 | 한국 국가 중 가장 이상한 법, 우리가 몰랐던 한글의 비밀 |
| 인체 잡지식 | 하루에 머리카락이 빠지는 개수, 눈썹이 없으면 생기는 일 |

## 3. 수집 소스 (우선순위 순)

### 1순위 — 학술·공공 기관
- 네이버 지식백과 (과학·역사 분야)
- 위키피디아 (한국어판 + 영어판 교차 확인)
- 국립국어원 (언어·어원 관련)
- 국사편찬위원회 (한국 역사 잡지식)
- 국립과학관 (과학 잡지식)

### 2순위 — 검증된 대중 콘텐츠
- TED-Ed 팩트 기반 영상
- BBC Science Focus
- 사이언스타임즈 (sciencetimes.co.kr)
- 국내 검증된 잡학 책 (알아두면 쓸데없는 신비한 잡학사전 류)

### 3순위 — 보조 참고 (반드시 1·2순위 교차 검증 필수)
- 유명 잡학 유튜브 채널 (긱블, 1분과학 등) 에서 언급된 팩트
- Quora, Reddit의 "Did you know" 스레드 — 반드시 출처 추적 후 사용
- 해외 trivia 전문 사이트 (mentalfloss.com, factretriever.com)

### 절대 사용 금지
- 출처 추적 불가능한 카카오톡 지식 공유 게시물
- "~카더라", "~라는 설이 있다" 수준의 불확실한 정보
- 음모론 계열 정보

## 4. 수집 필터 규칙

### 포함 기준
- 사실 여부 2개 이상의 독립 출처에서 교차 검증 가능
- "신기하다!"는 반응이 나올 만한 반전 or 의외성 포함
- 1분 내외로 설명 가능한 분량의 지식
- 시청자 일상생활과 연결되거나 연상 가능한 지식

### 제외 기준
- 확인 불가능한 "~카더라" 수준의 정보
- 특정 인종·집단에 대한 편견이 될 수 있는 정보
- 너무 전문적이어서 흥미 유발 어려운 정보
- 이미 널리 알려진 너무 유명한 상식 (예: "지구는 둥글다")
- 정치적으로 민감한 현재 이슈와 연결된 역사 정보

## 5. 영상 1편 구성 규칙

| 항목 | 규정 |
|------|------|
| 영상 길이 | 1분 ~ 2분 |
| 지식 개수 | 1편 = 1가지 지식 (깊게) OR 3~5가지 지식 (짧게 모음) |
| 구성 흐름 (단독형) | 훅 질문(10초) → 의외의 사실 공개(10초) → 이유 설명(40~60초) → 요약+여운(10초) |
| 구성 흐름 (모음형) | 훅(5초) → 지식 1~5 (각 15~20초) → 마무리(5초) |
| 제목 형식 | `알고 나면 쓸 데 없는 {주제} 잡지식 \| 몰랐던 사실` 또는 `[잡지식] 왜 {현상}일까?` |
| 썸네일 | 의문부호(?) + 핵심 키워드 + 임팩트 있는 숫자 or 이미지 |

## 6. 스크립트 작성 규칙

- 시작은 반드시 "질문형 훅"으로 시작 (예: "혹시 문어 심장이 몇 개인지 아세요?")
- 어투: 친구에게 신기한 얘기 해주는 가벼운 구어체
- 전문 용어 사용 시 즉시 쉬운 표현으로 바꿔 설명
- 중간에 "사실은요~", "놀랍게도~", "그런데 말이죠~" 등 리액션 어구 활용
- 마무리: 살짝 여운 남기기 ("이 사실 알고 나서 더 이상하게 보이지 않나요?")
- 출처 언급: 자막 또는 설명란에 반드시 포함
- 금지 표현: "100% 확실히", "절대적으로" 등 단정 표현 (사실도 틀릴 수 있으므로 "알려진 바에 따르면" 수준 유지)

## 7. 영상 프롬프트 생성 규칙

```
유형: 잡지식
주제: {지식 주제}
핵심 팩트: {핵심 내용 요약}
배경 스타일: 다크 배경 + 강렬한 타이포그래피, 정보 그래픽 스타일
색상: 네이비/블랙 배경 + 노란색·청록색 강조 텍스트
영상 요소: 키워드 텍스트 팝업, 간단한 인포그래픽, 연관 이미지
텍스트 스타일: 임팩트 있는 고딕, 숫자 크게 강조, 밑줄·원 강조 효과
나레이션 톤: 신기함과 흥미를 유발하는 약간 과장된 톤, 빠르고 리드미컬
배경음악: 흥미롭고 경쾌한 electronic pop (Mubert: curious quiz 태그)
전환 효과: 임팩트 있는 빠른 컷, 텍스트 팝업 애니메이션
훅 구성: 첫 3초 안에 "왜?/정말?" 반응 유도 필수
```

## 8. 중복 방지 규칙

- 메인 에이전트는 `잡지식_사용목록.json`으로 이미 사용된 지식 관리
- 동일 지식 재사용 금지 (최소 1년 이후 재사용 가능, 각색 필수)
- 유사 카테고리 연속 2편 이상 금지 (다양성 유지)

## 9. 품질 체크리스트

- [ ] 2개 이상 출처에서 사실 교차 검증 되었는가
- [ ] 첫 5초 안에 흥미 유발 훅이 있는가
- [ ] 전문 용어 없이 누구나 이해 가능한가
- [ ] 동일 지식이 이전에 사용된 적 없는가
- [ ] 특정 집단 편견 조장 내용이 없는가
- [ ] 영상 길이가 1~2분 범위인가
- [ ] 출처가 설명란에 포함 예정인가
