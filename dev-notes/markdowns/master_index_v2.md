## 4. 하루 제작 흐름

```
Step 1  GitHub Actions 트리거 (매일 오전 9시)
Step 2  메인 에이전트: master_index.md 읽고 오늘 제작 채널 결정
        → 검토대기 3편 이상 쌓이면 신규 제작 중단
Step 3  시나리오 에이전트: 해당 채널 시나리오룰.md 기반 수집 + 스크립트 작성
Step 4  이미지 에이전트: 프롬프트 → DALL-E 3 이미지 생성 (8~10장)
Step 5  나레이션 에이전트: 스크립트 → ElevenLabs TTS 생성
Step 6  FFmpeg 합성: 이미지 + 나레이션 + BGM → MP4 출력
Step 7  outputs/{채널}/검토대기/ 폴더 저장
Step 8  메인 에이전트: 이메일 알림 발송 (미리보기 링크 + 채널명 + 제목)
Step 9  사용자 검토 → 승인 / 수정요청 / 반려 회신
Step 10 승인 시: outputs/승인완료/ 이동 → YouTube 업로드 대기
        수정 시: 피드백 반영 재생성
        반려 시: outputs/반려/ 이동 + 실패 패턴 logs 기록
```
