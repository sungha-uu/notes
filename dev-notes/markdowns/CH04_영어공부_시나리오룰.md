# CH03 — 1분 영어 (영어 공부 콘텐츠)
> 채널 코드: `CH03_ENGLISH`  
> 채널 후보명: "1분 영어" / "매일 1분 영어" / "오늘의 영어 1분"  
> 대상 시청자: 영어 학습자 전 연령 (중학생 ~ 성인)  
> 영상 길이: 55초 ~ 1분 10초  
> 업로드 주기: 주 3회  
> 이미지 장수: 영상 1편당 8~10장

---

## 1. 콘텐츠 커리큘럼 및 우선순위

### 시즌 구조 (순서대로 진행)
```
시즌 1 — 영문법 기초     : 주어/동사/목적어, 시제, 조동사 등 핵심 문법 30편
시즌 2 — 핵심 영단어     : 빈출 단어 1000개 분류별 제공 (상황별 묶음)
시즌 3 — 숙어 & 이디엄   : "break a leg", "hit the sack" 등 실생활 숙어
시즌 4 — 자주 쓰는 표현  : 회화, 이메일, SNS에서 쓰는 표현 패턴
```

### 카테고리 코드

| 코드 | 카테고리 | 설명 | 월 편수 |
|------|----------|------|---------|
| `GRAM` | 영문법 | 헷갈리는 문법 규칙 1개 집중 설명 | 4편 |
| `WORD` | 영단어 | 상황별 단어 묶음 (5~7개) | 4편 |
| `IDIOM` | 숙어 | 숙어 1~2개 + 예문 3개 | 3편 |
| `EXPR` | 자주 쓰는 표현 | 상황별 표현 패턴 + 응용 | 3편 |

## 3. 카테고리별 스크립트 구성

### GRAM — 영문법
```
구성 흐름:
  00~10초 : 훅 — 많이 틀리는 상황 제시
             예) "이 두 문장, 차이 아세요?"
  10~20초 : 오개념 / 헷갈리는 포인트 제시
  20~50초 : 정확한 규칙 설명 + 예문 2~3개
  50~60초 : 핵심 규칙 한 줄 요약 + 오늘의 문장

필수 포함 :
  - 한국인이 특히 많이 틀리는 포인트 언급
  - 예문은 반드시 한국어 해석 병기
  - "오늘의 핵심" 한 줄 요약 마무리

예시 주제 :
  - "I have been" vs "I have" 차이
  - "Make" vs "Do" 구분법
  - "some" vs "any" 언제 쓰나
```

### WORD — 영단어
```
구성 흐름:
  00~08초 : 오늘의 테마 소개 ("오늘은 감정 표현 단어예요")
  08~50초 : 단어 5~7개 (단어 → 발음 → 뜻 → 예문 순서)
  50~60초 : 전체 단어 빠른 복습

필수 포함 :
  - 단어마다 실생활 예문 1개
  - 발음 강조 (헷갈리기 쉬운 단어는 발음 주의 언급)
  - 단어 묶음 테마 일관성 유지

테마 예시 :
  - 감정 표현 단어, 음식 관련 단어, 직장 생활 단어
  - 여행에서 쓰는 단어, SNS에서 자주 보는 단어
```

### IDIOM — 숙어
```
구성 흐름:
  00~10초 : 숙어를 직역하면 이상한 상황 제시 (유머 포인트)
             예) "Break a leg? 다리를 부러뜨리라고요?"
  10~40초 : 실제 의미 + 유래 + 예문 2~3개
  40~60초 : 비슷한 표현 or 반대 표현 1개 보너스

필수 포함 :
  - 직역 vs 실제 의미 대비 (임팩트 포인트)
  - 어떤 상황에서 쓰는지 context 명시
  - 너무 격식체 or 구어체인 경우 레이블 명시
```

### EXPR — 자주 쓰는 표현
```
구성 흐름:
  00~08초 : 이 표현이 필요한 상황 제시
             예) "해외에서 음식 추가 요청할 때 뭐라고 해요?"
  08~45초 : 핵심 표현 1~2개 + 응용 표현 2~3개
  45~60초 : 오늘의 표현 빠른 정리

필수 포함 :
  - 상황(context) 먼저 제시 후 표현 소개
  - 원어민이 실제로 쓰는 자연스러운 표현 우선
  - 격식/비격식 구분 명시
```

## 4. 공통 스크립트 규칙

```
언어 혼용  : 한국어 설명 + 영어 예문 병기 (한국어 70% / 영어 30%)
발음 표기  : 영어 단어 첫 등장 시 한국어 발음 표기 병행
              예) "Inevitable [이네비터블]"
문장 길이  : TTS 1문장 = 20자 이내 권장
톤         : 진지하되 딱딱하지 않게 — 친절한 선생님 느낌
             "어렵지 않아요, 이렇게 생각하면 돼요" 느낌
반복 강조  : 핵심 표현은 영상 내 최소 2회 반복 노출 (기억 유도)
마무리     : "오늘 배운 표현, 오늘 한 번만 써보세요!" 고정 멘트
```

```
Korean educational content aesthetic, clean whitespace composition
```

### 장면별 구성 (1편 = 9장 기준)
```
01장 : 제목 카드 — 오늘의 주제 텍스트 삽입용 밝은 배경 여백
02장 : 훅 장면 — 상황 묘사 (캐릭터가 영어로 말하는 상황)
03장 : 오개념/헷갈리는 상황 — X 표시와 함께
04~07장 : 핵심 설명 장면 — 단어/문법/예문 텍스트 삽입용 공간
08장 : 핵심 정리 카드 — 오늘의 요약 텍스트 삽입용 강조 배경
09장 : 마무리 카드 — "오늘 한 번 써보세요!" 고정 문구 삽입용
```

### 카테고리별 분위기 조정

| 카테고리 | 배경 | 색상 포인트 |
|----------|------|-------------|
| GRAM | 노트/칠판 느낌 깔끔한 배경 | 화이트 + 연파랑 + 연노랑 |
| WORD | 밝고 생동감 있는 상황별 배경 | 화이트 + 민트 + 연살구 |
| IDIOM | 살짝 유머러스한 상황 묘사 | 화이트 + 연노랑 + 연보라 |
| EXPR | 실생활 장면 (카페, 공항, 직장) | 화이트 + 연파랑 + 연초록 |

### 이미지 내 텍스트 삽입 공간 (필수)
```
- 모든 이미지는 영어 단어 / 문법 / 예문 텍스트가 삽입될 공간 확보
- 이미지 하단 또는 중앙에 흰색/연한 배경 박스 영역 필수
- 영어 텍스트는 FFmpeg 자막으로 삽입 (이미지에 직접 쓰지 않음)
```

### 금지 이미지 요소
```
- 실제 교과서·시험지 이미지 (저작권)
- 특정 국가 국기를 부정적 맥락으로 사용
- 어둡거나 위협적인 분위기
- 복잡하고 어지러운 구도 (교육 콘텐츠이므로 깔끔함 유지)
```

## 6. 나레이션 설정 (ElevenLabs)

```
언어       : 한국어 (영어 예문 구간은 영어 발음으로 자연스럽게)
목소리 톤  : 밝고 또렷한 여성 or 남성 목소리 (친절한 선생님 느낌)
속도       : 설명 구간 0.95x (또렷하게)
             영어 단어/표현 읽을 때 0.85x (정확한 발음 전달)
강조 처리  : 핵심 영어 표현은 살짝 강조해서 읽기
```

## 7. 배경음악 규칙

```
소스        : YouTube Audio Library / Pixabay Music
기본 무드   : 밝고 집중되는 경쾌한 어쿠스틱 or 로파이 (BPM 85~100)
학습 집중   : 너무 신나거나 산만하지 않게 — 카페 배경음악 느낌
볼륨 믹싱   : 나레이션 100%, 배경음악 15~20% (영어 학습이므로 말이 잘 들려야 함)
금지        : 가사 있는 음악, 템포 빠른 댄스 음악
```

## 8. 자막 규칙 (FFmpeg 삽입)

```
영어 표현  : 화면 중앙 또는 상단 — 굵은 폰트, 흰색 + 검정 테두리
한국어 해석: 영어 아래 — 작은 크기, 연회색
핵심 강조  : 중요 단어에 노란색 하이라이트 효과
발음 표기  : 영어 단어 옆에 괄호로 한국어 발음 표기
             예) inevitable [이네비터블]
```

## 9. 제목 & 썸네일 공식

```
제목 형식  :
  패턴 A — "[1분 영어] {표현/단어} | 한국인이 {빈도} 틀리는"
  패턴 B — "오늘의 영어 1분 | {주제} {숫자}가지"
  패턴 C — "{영어 표현}? 이렇게 쓰면 됩니다 | 1분 영어"

예시       :
  - [1분 영어] make vs do | 한국인이 가장 많이 틀리는 동사
  - 오늘의 영어 1분 | 감정 표현 단어 7가지
  - "Break a leg"? 이게 무슨 뜻? | 영어 숙어 1분

썸네일 구성 :
  - 배경: 밝은 흰색 or 연파랑
  - 중앙: 핵심 영어 표현 크게 (굵은 폰트)
  - 한국어 의미 작게 아래 배치
  - 포인트: 노란색 하이라이트 or 강조선
  - 캐릭터: 밝게 웃는 선생님 or 학생 일러스트
```
