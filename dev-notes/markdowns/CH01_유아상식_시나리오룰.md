# CH01 — 유아 상식과 팁
> 채널 코드: `CH01_BABY`  
> 대상 시청자: 0~36개월 아이를 키우는 부모  
> 영상 길이: 50초 ~ 1분 20초  
> 업로드 주기: 주 3회  
> 이미지 장수: 영상 1편당 8~12장

---

## 1. 콘텐츠 카테고리 및 우선순위

| 코드 | 카테고리 | 설명 | 월 편수 |
|------|----------|------|---------|
| `SICK` | 아플 때 대처법 | 열, 기침, 구토, 설사, 발진 등 증상별 대처 | 4편 |
| `MEDS` | 먹여도 되는 약 | 개월수별 복용 가능 약 / 용량 / 주의사항 | 3편 |
| `FOOD` | 유아식 조리법 | 이유식, 간편 유아식, 핑거푸드, 간식 레시피 | 5편 |
| `TIPS` | 월령별 육아 팁 | 수면, 놀이, 발달, 훈육, 생활 습관 | 4편 |
| `BOOK` | 책 & 놀이 추천 | 개월수별 그림책, 놀이 방법, 장난감 추천 | 2편 |

---

## 2. 개월수 구간 정의

```
신생아기  : 0 ~ 3개월
초기 영아 : 4 ~ 6개월
중기 영아 : 7 ~ 9개월
후기 영아 : 10 ~ 12개월
초기 유아 : 13 ~ 18개월
중기 유아 : 19 ~ 24개월
후기 유아 : 25 ~ 36개월
```
- 영상 제목과 썸네일에 개월수를 항상 명시
- 1편 = 1개월수 구간 + 1카테고리 조합으로 제작

## 3. 정보 수집 소스

### 필수 1순위 (의료·공공기관)
```
- 대한소아과학회         : pediatrics.or.kr
- 질병관리청             : kdca.go.kr
- 보건복지부 아이사랑    : childcare.go.kr
- 서울대학교병원 소아과  : snuh.org
```

### 2순위 (전문가 검증 콘텐츠)
```
- 소아과 전문의 유튜브 (구독자 10만 이상)
- 국내 유아 전문 매거진 (베이비뉴스, 키즈맘)
- WHO 및 AAP 한국어 번역 자료
```

### 수집 제외 기준
```
- 출처 불명 SNS / 맘카페 단독 출처
- 5년 이상 된 구식 의료 정보
- 특정 브랜드 홍보 목적 콘텐츠
- "100% 효과" 등 과장 표현 포함 자료
```

## 4. 카테고리별 스크립트 규칙

### SICK / MEDS — 아플 때 · 약
```
문장 톤    : 침착하고 신뢰감 있게. 지나치게 밝지 않음.
              "걱정되실 텐데, 이렇게 해보세요" 느낌
구성 흐름  : 증상 제시(10초) → 대처법 2~3가지(40초) → 병원 가야 할 신호(10초)
필수 문구  : "개인차가 있으니 증상이 심하면 소아과를 방문하세요"
금지 표현  : "먹이면 바로 낫습니다", "걱정 안 해도 됩니다" 등 단정
수치 필수  : 약 용량은 반드시 kg당 용량 또는 개월수 기준 수치 명시
```

### FOOD — 유아식 조리법
```
문장 톤    : 친근하고 따뜻하게. "오늘 뭐 해줄까?" 느낌
구성 흐름  : 완성 음식 장면(5초) → 재료 소개(15초) → 조리 순서(30초) → 보관 팁(10초)
분량 기준  : 기본 2인분 (아이 1 + 보호자 1), 첫 장면에 인분 수 명시
수치 필수  : "소금 조금" 금지 → "소금 1/4 티스푼" 방식으로 명시
필수 포함  : 알레르기 유발 식품 경고 (달걀, 땅콩, 우유 등 포함 시)
```

### TIPS / BOOK — 팁 & 추천
```
문장 톤    : 밝고 명랑하게. 엄마 친구한테 얘기해주는 느낌
구성 흐름  : 훅 질문(10초) → 팁 3~5가지(50초) → 마무리 한 줄(10초)
팁 개수    : 1편당 3~5가지
책 추천 시 : 제목 + 추천 이유 한 줄 + 적정 월령 반드시 포함
```

## 5. 공통 스크립트 규칙

```
호칭        : "아기", "우리 아이" 혼용 (같은 단어 3회 이상 연속 금지)
문장 길이   : 나레이션 1문장 = 15자 이내 권장 (TTS 호흡 고려)
금지 표현   : "무조건", "절대로", "100% 효과", "완치"
필수 마무리 : 영상 마지막 장면에 항상 포함
              → "더 궁금한 점은 소아과 선생님께 물어보세요 :)"
전문 용어   : 사용 시 바로 다음 문장에 쉬운 설명 병기
              예) "모유 수유 텀이 → 젖을 먹이는 간격이"
```

## 6. 이미지 생성 규칙 (DALL-E 3 프롬프트 패턴)

### 공통 베이스 프롬프트
```
Style base: Soft pastel illustration, rounded cute characters,
warm ivory and mint background, gentle lighting, no harsh shadows,
flat design with slight depth, Korean children's book aesthetic,
clean and minimal composition
```

### 카테고리별 분위기 조정

| 카테고리 | 분위기 조정값 | 색상 포인트 |
|----------|-------------|-------------|
| SICK / MEDS | `slightly serious tone, softer muted pastels, gentle concerned expression` | 연보라, 연회색, 민트 |
| FOOD | `cheerful bright tone, warm kitchen setting, colorful food props` | 연노랑, 연살구, 민트 |
| TIPS / BOOK | `bright and playful, happy baby characters, soft sunlight` | 연두, 연파랑, 연노랑 |

### 이미지 구성 순서 (1편 = 10장 기준)
```
01장 : 제목 카드 (개월수 + 주제 텍스트 삽입용 여백 있는 배경)
02장 : 훅 장면 (아기 캐릭터 + 상황 묘사)
03~08장 : 본문 내용 장면 (핵심 팁 또는 조리 과정)
09장 : 강조 포인트 카드 (핵심 메시지 텍스트 삽입용)
10장 : 마무리 카드 ("소아과 선생님께 물어보세요" 문구 삽입용 여백)
```

### 금지 이미지 요소
```
- 주사기, 피, 상처 직접 묘사
- 울고 있는 아기 단독 장면 (보호자와 함께라면 가능)
- 실사 사진 스타일 (일러스트만 허용)
- 어둡거나 무서운 색조
```

## 7. 나레이션 설정 (ElevenLabs)

```
언어       : 한국어
목소리 톤  : 따뜻하고 부드러운 여성 목소리
속도       : 약간 느린 편 (0.9x) — 육아 정보 전달이라 또렷하게
SICK/MEDS  : 속도 0.85x, 톤 차분하게
FOOD/TIPS  : 속도 0.95x, 톤 밝고 친근하게
```

## 8. 배경음악 규칙

```
소스        : YouTube Audio Library / Pixabay Music
SICK/MEDS  : 잔잔한 피아노 or 어쿠스틱 기타 (BPM 60~70)
FOOD       : 경쾌하고 따뜻한 어쿠스틱 (BPM 90~100)
TIPS/BOOK  : 밝고 귀여운 동요풍 멜로디 (BPM 80~90)
볼륨 믹싱  : 나레이션 100%, 배경음악 20~25%
```

## 9. 제목 & 썸네일 공식

```
제목 형식  : [N개월 아기] {핵심 키워드} {숫자}가지 | {부제}
예시       :
  - [6개월 아기] 열날 때 대처법 3가지 | 소아과 가야 할 신호까지
  - [8개월 아기] 첫 핑거푸드 레시피 5가지 | 10분 완성
  - [12개월 아기] 이 시기 꼭 읽어줘야 할 그림책 추천

썸네일 구성 :
  - 배경: 파스텔 톤 (민트 or 연노랑)
  - 중앙: 아기 일러스트 캐릭터
  - 상단: 개월수 뱃지 (크게)
  - 하단: 핵심 키워드 2~3단어 (굵은 폰트)
```

## 10. 품질 체크리스트

```
[ ] 개월수 구간이 제목과 영상 첫 장면에 명시되었는가
[ ] 의료 정보 출처가 1순위 소스로 확인되었는가
[ ] 소아과 상담 권고 문구가 마지막 장면에 포함되었는가
[ ] 과장 표현이 없는가
[ ] 이미지 10장 순서가 흐름에 맞게 구성되었는가
[ ] SICK/MEDS 카테고리는 분위기가 과도하게 밝지 않은가
[ ] 나레이션 1문장이 15자 이내인가
[ ] 배경음악 볼륨이 나레이션보다 작은가
[ ] 알레르기 유발 식품 경고가 FOOD 콘텐츠에 포함되었는가
[ ] 영상 길이가 50초 ~ 1분 20초 범위인가
```
