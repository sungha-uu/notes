# CH02 — 몰라도 되는데 궁금한 (잡지식)
> 채널 코드: `CH02_TRIVIA`  
> 채널 후보명: "몰라도 되는데 궁금한" / "어? 진짜요?"  
> 대상 시청자: 15~45세 전 연령 (지식 호기심층)  
> 영상 길이: 50초 ~ 1분 10초  
> 업로드 주기: 주 3~4회  
> 이미지 장수: 영상 1편당 8~10장

---

## 1. 콘텐츠 카테고리

| 코드 | 카테고리 | 예시 주제 | 월 편수 |
|------|----------|----------|---------|
| `SCI` | 과학 잡지식 | 왜 종이는 7번 이상 못 접나, 파란 눈의 기원 | 4편 |
| `HIST` | 역사 잡지식 | 나폴레옹 키의 진실, 로마의 치약은 소변이었다 | 3편 |
| `LANG` | 언어 & 어원 | "바보"의 어원, 영어에서 가장 긴 단어 | 3편 |
| `BIO` | 동물 & 인체 | 문어 심장 3개, 달팽이 이빨 2만 개 | 4편 |
| `LIFE` | 일상의 비밀 | 하품이 전염되는 이유, 팝콘이 비싼 진짜 이유 | 3편 |
| `KOR` | 한국 잡지식 | 한글의 숨겨진 원리, 우리가 몰랐던 한국 법 | 3편 |

## 2. 분위기 레퍼런스

```
핵심 분위기 키워드:
  - TV 프로그램 "서프라이즈" — 반전과 충격, 진지한 나레이션
  - TV 프로그램 "그것이 알고싶다" — 궁금증 유발, 긴장감 있는 전개
  - 유튜브 "사물궁이" — 친근하게 파고드는 설명
  - 유튜브 "1분과학" — 빠르고 임팩트 있는 전달

한 줄 요약:
  "진지하게 물어보는데 알고 보면 쓸데없는" 느낌
  → 처음엔 흥미롭게 끌어들이고, 정답을 공개하면서 "어? 진짜?" 반응 유도
```

## 3. 정보 수집 소스

### 1순위 — 학술·공공기관 (교차 검증 필수)
```
- 네이버 지식백과 (과학/역사 분야)
- 위키피디아 한국어판 + 영어판 교차 확인
- 국립국어원 (언어·어원)
- 국사편찬위원회 (한국 역사)
- 사이언스타임즈 : sciencetimes.co.kr
```

### 2순위 — 검증된 대중 콘텐츠
```
- TED-Ed 팩트 기반 영상
- BBC Science Focus
- Mental Floss : mentalfloss.com
- 국내 잡학 서적 (알쓸신잡 시리즈 등) — 내용 참고, 그대로 인용 금지
```

### 수집 제외 기준
```
- 단일 출처만 확인된 정보 (2개 이상 교차 검증 필수)
- "~카더라" 수준의 루머성 정보
- 음모론 / 근거 없는 민간 속설
- 특정 인종·종교·집단 비하로 해석될 수 있는 소재
- 이미 너무 유명한 상식 (지구는 둥글다 수준)
```

## 4. 스크립트 구성 규칙

### 영상 흐름 (총 1분 기준)
```
00~10초 : 훅 — 질문형으로 시작, 반드시 의문 유발
           예) "혹시 문어 심장이 몇 개인지 아세요?"
           예) "나폴레옹이 사실 키가 작지 않았다고요?"

10~15초 : 일반적인 통념 제시 (시청자가 알고 있다고 생각하는 것)
           예) "대부분 아주 작다고 알고 있는데요..."

15~50초 : 진실 공개 + 이유 설명
           - 사실을 밝히고 → 왜 그렇게 됐는지 → 근거 제시
           - "사실은요~", "놀랍게도~", "그런데 말이죠~" 활용

50~60초 : 여운 + 마무리
           예) "이 사실 알고 나서 문어 볼 때마다 생각날 것 같죠?"
```

### 톤 & 문체
```
기본 톤   : 진지하게 시작 → 반전 공개 시 약간 흥분
나레이션  : 마치 비밀을 알려주는 듯한 속도감 있는 전달
문장 스타일: 구어체, 짧고 끊어서 ("그런데요." / "아닙니다." / "바로~")
금지 표현 : "100% 확실히", "절대적으로" — "알려진 바로는", "연구에 따르면"으로 대체
출처 명시 : 영상 하단 자막 또는 설명란에 반드시 포함
```

### 중복 방지
```
사용된 지식은 logs/CH02_사용목록.json에 기록
동일 지식 재사용: 최소 12개월 후 가능 (각색 필수)
같은 카테고리 연속 3편 금지 (다양성 유지)
```

## 5. 이미지 생성 규칙 (DALL-E 3 프롬프트 패턴)

### 공통 베이스 프롬프트
```
Style base: Dark mysterious background (deep navy or dark gray),
bold dramatic typography style, high contrast lighting,
cinematic documentary feel, slightly noir atmosphere,
clean infographic elements, Korean mystery show aesthetic,
no horror elements, sophisticated and intriguing mood
```

### 장면별 구성 (1편 = 9장 기준)
```
01장 : 의문 카드 — 어두운 배경 + 물음표 + 핵심 질문 텍스트 삽입용 여백
02장 : 통념 장면 — 일반적으로 알려진 잘못된 상식 묘사
03장 : 반전 예고 카드 — "사실은..." 텍스트 삽입용 여백 + 긴장감 있는 구도
04~07장 : 진실 설명 장면 — 근거와 이유를 시각적으로 묘사
08장 : 핵심 팩트 강조 카드 — 임팩트 있는 숫자/키워드 텍스트 삽입용
09장 : 마무리 여운 카드 — 시청자에게 질문 던지는 구도
```

### 카테고리별 분위기 조정

| 카테고리 | 배경 분위기 | 색상 포인트 |
|----------|-----------|-------------|
| SCI | 우주·실험실·현미경 느낌 | 딥 네이비 + 청록 |
| HIST | 오래된 양피지·박물관 느낌 | 다크 브라운 + 골드 |
| LANG | 타이포그래피 중심 | 차콜 + 화이트 |
| BIO | 자연·생물 다큐 느낌 | 딥 그린 + 앰버 |
| LIFE | 도시·일상의 비하인드 | 다크 그레이 + 레드 포인트 |
| KOR | 한국 전통 + 현대 믹스 | 다크 네이비 + 금색 |

### 금지 이미지 요소
```
- 실제 인물 얼굴 사진 (일러스트 or 실루엣만 허용)
- 혐오감 유발 이미지 (벌레 클로즈업 등)
- 폭력·유혈 묘사
- 특정 종교 상징물 직접 묘사
```

## 6. 나레이션 설정 (ElevenLabs)

```
언어       : 한국어
목소리 톤  : 중저음의 신뢰감 있는 남성 목소리 (다큐 나레이터 느낌)
              또는 또렷하고 차분한 여성 목소리
속도       : 훅 구간 0.95x → 설명 구간 1.0x → 반전 공개 시 살짝 빠르게 1.05x
강조 처리  : 핵심 단어는 약간 끊어 읽기 ("바로... 3개입니다!")
```

## 7. 배경음악 규칙

```
볼륨 믹싱   : 나레이션 100%, 배경음악 18~22%
금지        : 너무 밝은 팝, 코믹한 효과음, 공포 영화풍 음악
```

## 8. 제목 & 썸네일 공식

```
제목 형식  :
  패턴 A — "{충격적 사실} | 몰랐던 진짜 이유"
  패턴 B — "왜 {현상}일까? | 알고 나면 소름"
  패턴 C — "{통념} 은 사실이 아니었다 | {진실}"

예시       :
  - 나폴레옹은 사실 작지 않았다 | 오해가 생긴 진짜 이유
  - 왜 하품은 전염될까? | 알고 나면 소름
  - 문어 심장이 3개인 이유 | 진화의 비밀

썸네일 구성 :
  - 배경: 다크 네이비 or 차콜 그레이
  - 중앙: 핵심 이미지 (강렬한 조명 연출)
  - 상단: 물음표(?) 또는 "진짜?" 텍스트
  - 하단: 핵심 키워드 굵은 흰색 폰트
  - 포인트: 노란색 or 청록색 강조선
```
