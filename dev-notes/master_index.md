## 3. 일일 스케줄링 규칙

### 기본 원칙
- 하루 제작 목표: 최대 5편 (승인 부담 최소화)
- 채널 균형: 동일 채널 하루 2편 이상 제작 금지
- 우선순위: 검토 대기 영상이 3편 이상 쌓이면 신규 제작 일시 중단

### 요일별 채널 배정 (예시)
| 요일 | 제작 채널 |
|------|----------|
| 월 | 잡지식 × 2, 육아팁 × 1 |
| 화 | 요리레시피 × 2, 전래동화 × 1 |
| 수 | 맛집(지역별) × 1, 당뇨식단 × 1, 잡지식 × 1 |
| 목 | 육아팁 × 1, 전래동화 × 1, 자장가 × 1 |
| 금 | 맛집(음식별) × 1, 요리레시피 × 1, 잡지식 × 1 |
| 토 | 당뇨식단 × 1, 전래동화 × 1 |
| 일 | 자장가 × 1, 예비 재작업 처리 |

## 4. 사용자 승인 프로세스

```
[제작 완료]
    ↓
메인 에이전트 → 이메일 + 카카오톡 알림 발송
(영상 미리보기 링크 + 제목 + 채널명 포함)
    ↓
사용자 피드백 (24시간 이내 응답 권장)
    ↓
┌─────────────────────────────┐
│ 승인    → outputs/승인완료/ 이동 → YouTube 업로드 대기 or 즉시 업로드
│ 수정요청 → outputs/수정중/ 이동  → 피드백 반영 재생성
│ 반려    → outputs/반려/ 이동    → 실패 패턴 logs 기록
└─────────────────────────────┘
```

## 5. 일일 보고서 포함 항목

```markdown
## [날짜] 일일 제작 보고서

### 오늘 제작 현황
- 제작 완료: N편
- 채널별: 육아팁 N편, 맛집 N편, ...

### 승인 대기 현황
- 대기 중: N편 (가장 오래된 대기: N일)

### 이번 주 누적
- 제작: N편 / 목표 21편
- 승인율: N%

### 내일 예정
- 제작 채널: {채널명}
- 예정 주제: {주제 목록}

### 알림
- 특이사항 또는 수정 필요 사항
```

## 6. 채널별 특이사항 메모

| 채널 | 주의사항 |
|------|---------|
| 당뇨식단 | 의료 정보 포함 — 면책 고지 필수, 출처 반드시 명시 |
| 유아 자장가 | 음원 저작권 사전 확인 필수 — 무단 사용 시 유튜브 수익화 차단 위험 |
| 육아팁 | 소아과 전문의 상담 권고 문구 필수 |
| 맛집 | 지역별·음식별 동일 식당 2개월 내 중복 등장 금지 |
| 잡지식 | 사실 2개 이상 출처 교차 검증 필수, 사용 목록 로그 관리 |
| 전래동화 | 저작권 소멸 전래동화만 사용, 각색 시 원전 출처 명시 |
