# 지역별 맛집 시나리오룰.md
> 채널: 맛집 유튜브 채널 | 유형 코드: `RESTAURANT_REGION` | 시나리오 파일: A (지역별)

---

## 1. 콘텐츠 목적
전국 각 지역의 믿을 수 있는 맛집을 큐레이션하여,
여행·나들이·출장 시 "이 지역에서 어디가 진짜 맛있나?"라는 질문에 바로 답해주는 콘텐츠를 만든다.

---

## 2. 지역 분류 체계

### 대분류
| 권역 | 포함 지역 |
|------|----------|
| 서울 | 강남, 홍대, 이태원, 종로, 을지로, 성수, 마포 등 구별 세분화 |
| 경기/인천 | 수원, 분당, 일산, 인천 차이나타운, 강화도 등 |
| 강원 | 속초, 강릉, 춘천, 평창, 태백 등 |
| 충청 | 대전, 청주, 천안, 공주, 부여 등 |
| 전라 | 전주, 광주, 여수, 순천, 목포 등 |
| 경상 | 부산, 대구, 경주, 안동, 통영, 거제 등 |
| 제주 | 제주시, 서귀포, 애월, 성산 등 세분화 |

### 세분화 기준
- 서울은 구(區) 단위까지 세분화 (예: 마포구 맛집, 성동구 맛집)
- 지방 광역시는 구 단위, 소도시는 도시 단위
- 관광지 특화 지역은 별도 편성 (예: 전주 한옥마을, 경주 황리단길)

---

## 3. 수집 소스 (우선순위 순)

### 1순위 — 검증된 리뷰 플랫폼
- 네이버 지도 (별점 4.0 이상 + 리뷰 200개 이상)
- 카카오맵 (별점 4.0 이상 + 리뷰 100개 이상)
- 망고플레이트 (전문 에디터 픽 우선)
- 식신 (지역 맛집 랭킹 상위)

### 2순위 — 공신력 있는 미디어
- 한국관광공사 추천 맛집 (visitkorea.or.kr)
- 지역 신문사 맛집 기사 (부산일보, 전남일보 등)
- 유명 맛집 유튜버 방문 후기 (100만 이상 구독자)
- 미쉐린 가이드 서울 / 빕 구르망 선정 식당

### 3순위 — 보조 참고
- 인스타그램 해시태그 (#지역명맛집) — 최근 6개월 이내 게시물
- 블로그 체험단 후기 — 광고 표기 여부 반드시 확인, 순수 후기만 채택
- ※ 블로그·SNS 단독 소스 사용 금지

---

## 4. 수집 필터 규칙

### 포함 기준
- 최근 6개월 이내 영업 확인된 식당
- 네이버 or 카카오 별점 4.0 이상
- 리뷰 수 최소 50개 이상 (소도시는 30개 완화 적용)
- 현지인 + 여행자 리뷰 혼재된 곳 (한쪽만 치우친 경우 제외)
- 가격 정보 확인 가능한 곳

### 제외 기준
- 폐업 또는 휴업 중인 식당
- 광고성 블로그 포스팅만 있는 식당
- 별점 조작 의심 (리뷰 수 대비 별점 과도하게 높음)
- 프랜차이즈 대형 체인 (지역 특색 없는 곳)
- 최근 위생 문제 기사가 있는 식당

---

## 5. 영상 1편 구성 규칙

| 항목 | 규정 |
|------|------|
| 영상 길이 | 2분 ~ 3분 |
| 소개 맛집 수 | 1편당 5~7곳 |
| 구성 흐름 | 훅(10초) → 지역 소개(15초) → 맛집 리스트(90~120초, 곳당 15~20초) → 지도 정리(20초) → CTA(10초) |
| 제목 형식 | `[지역명] 현지인이 진짜 가는 맛집 N곳 \| 주소 포함` |
| 썸네일 키워드 | 지역명 + 대표 음식 사진 |

---

## 6. 맛집 1곳당 수집 정보 항목

```
- 식당명 (한글 + 영문 병기)
- 대표 메뉴 1~2가지 + 가격
- 주소 (네이버 지도 링크 포함)
- 영업시간 + 정기 휴무일
- 주차 가능 여부
- 예약 필요 여부
- 특이사항 (웨이팅 필요, 현금만 가능 등)
- 추천 이유 한 줄 (예: "60년 전통 할머니 손맛 순대국밥")
```

---

## 7. 스크립트 작성 규칙

- 어투: 여행 유튜버 느낌의 경쾌하고 설레는 구어체
- 각 맛집마다 "왜 가야 하는지" 이유 한 줄 필수
- 지역 역사·문화 간단 언급으로 현지감 부여
- 금지 표현: "무조건 맛있다", "실망 없다" 등 무조건적 보증
- 필수 포함: "방문 전 영업시간 확인 필수" 문구
- 마지막에 지도 화면 보여주며 위치 정리

---

## 8. 영상 프롬프트 생성 규칙 (영상 에이전트 전달용)

```
유형: 맛집 (지역별)
지역: {지역명}
소개 맛집: {식당1}, {식당2}, ..., {식당N}
배경 스타일: 해당 지역 분위기 반영 (제주=파란하늘, 전주=한옥, 부산=바다)
영상 요소: 음식 사진 클로즈업, 거리 풍경 B-roll, 지도 UI
텍스트 스타일: 굵은 고딕, 식당명은 노란 배경 박스
나레이션 톤: 활기차고 식욕 자극하는 남성 또는 여성 목소리
배경음악: 경쾌한 어쿠스틱 or 재즈 (Mubert: travel food 태그)
전환 효과: 빠른 컷, 줌인 효과
말미 구성: 지도 UI로 맛집 위치 한 번에 표시
```

## 9. 품질 체크리스트 (메인 에이전트 검수용)

- [ ] 소개된 모든 식당 현재 영업 중인지 확인
- [ ] 주소·영업시간 정보 최신인지 확인
- [ ] 프랜차이즈 체인이 섞이지 않았는지 확인
- [ ] 별점 4.0 이상 기준 충족 여부
- [ ] 지역 특색이 영상 스타일에 반영되었는가
- [ ] 영상 길이 2~3분 범위인가
- [ ] 지도 정리 화면이 포함되었는가
